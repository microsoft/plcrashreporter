Detailed Description

A crash report formatter accepts a PLCrashReport instance, formats it according to implementation-specified rules, (such as implementing text output support), and returns the result.

Parameters
reportReport to be formatted.
outErrorA pointer to an NSError object variable. If an error occurs, this pointer will contain an error object indicating why the pending crash report could not be formatted. If no error occurs, this parameter will be left unmodified. You may specify nil for this parameter, and no error information will be provided.
Returns
Returns the formatted report data on success, or nil on failure.
